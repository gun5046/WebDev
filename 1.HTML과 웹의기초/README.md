### Introduction
------------------------------------------------------------------
HTML은 HyperText Markup Language의 약자로, 웹 브라우저에 내용을 표시하기 위한 가장 기본적인 언어입니다. HTML에 관한 기초적인 사항들을 알아볼 예정입니다.
<br/>
<br/>

  + XHTML 2.0은 왜 세상에 나오지 못하게 되었을까요?
